# Qwen Image Edit API - Python Demo

Demo notebook and script for using the Qwen Image Edit API.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Place your test image as `test.png` in this directory

3. Run the notebook:
```bash
jupyter notebook qwen_api_demo.ipynb
```

## Files

- `qwen_api_demo.ipynb` - Interactive Jupyter notebook
- `requirements.txt` - Python dependencies
- `test.png` - Your input image (you need to provide this)
- `result.png` - Output will be saved here

## API Details

- **Endpoint**: https://qwen.codeblazar.org
- **API Key**: Included in notebook (can be rotated with `.\api\manage-key.ps1 -Rotate`)
- **Documentation**: https://qwen.codeblazar.org/docs
